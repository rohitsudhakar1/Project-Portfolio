# Portfolio — rohitsudhakar1

Source for my portfolio landing page.

**Live:** https://rohitsudhakar1.github.io/Project-Portfolio/

## Stack

Hand-written HTML / CSS / JS. No framework, no build step. Hosted via GitHub Pages from `main`.

## Local preview

```bash
# any static server works
python -m http.server 8000
# then visit http://localhost:8000
```

## Files

- `index.html` — markup
- `styles.css` — all styles
- `script.js` — clock, build-id, card spotlight, scroll reveal

## Aesthetic

Warm-dark "engineer's notebook": editorial serif display (Fraunces) paired with JetBrains Mono, amber accent, subtle grain + grid lines. Built to feel handmade, not templated.
